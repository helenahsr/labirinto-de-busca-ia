01 · USO

Como utilizar

Escolha o algoritmo (A* ou Gulosa), ajuste o tamanho do labirinto e a velocidade. Clique em Executar Busca para ver o rato encontrar o queijo. Para um novo desafio, gere outro labirinto ou apenas reposicione o queijo.

02 · A*

Algoritmo A-estrela

Combina o custo real percorrido g(n) com a estimativa h(n) até o objetivo. Garante encontrar o caminho ótimo quando a estimativa é admissível.

f(n) = g(n) + h(n)
03 · GULOSA

Busca Gulosa

Considera apenas a estimativa h(n), escolhendo sempre o vizinho aparentemente mais próximo do objetivo. É rápida, mas pode não retornar o caminho mais curto.

f(n) = h(n)
04 · HEURÍSTICA

Distância de Manhattan

Soma das diferenças absolutas entre coordenadas. Ideal para grids com movimentos ortogonais (cima, baixo, esquerda, direita) — exatamente o caso deste labirinto.

h = |x₁−x₂| + |y₁−y₂|
05 · GERAÇÃO

Construção do labirinto

As paredes são geradas pelo algoritmo de recursive backtracking, que constrói um labirinto perfeito onde todas as células são alcançáveis a partir da origem — garantindo sempre um caminho até o queijo.

06 · CORES

O que cada cor significa

Azul = fronteira (próximos a visitar); cinza-azul = já explorados; âmbar = caminho ótimo encontrado.
